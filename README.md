# Personality Predictor

This is a Streamlit web app that predicts personality type using a trained logistic regression model.

## Features
- User-friendly Streamlit interface
- Personality prediction from input features
- Uses saved model and scaler files

## Files
- `app.py` - Streamlit application
- `personality_model.pkl` - trained model
- `scaler.pkl` - scaler used for preprocessing
- `requirements.txt` - required Python packages

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
